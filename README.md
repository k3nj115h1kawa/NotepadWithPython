# NotepadWithPython
## Pythonでメモ帳作成
### 行ったこと
* 「pythonでメモ帳を作ってみた。」を参照
* 上書き保存ボタンを設置してみた
* 上書き保存時にダイアログボックスを表示しないために, filedialog.asksavefilename関数ではなく, open関数を使用
* ファイル名を別途保持しておく
